### 客户概况

温州市国彩真空科技有限公司是浙南地区规模最大的PVD镀膜加工服务商之一，年加工能力约200万件，客户分布在渔具、餐具、五金、3C配件等多个行业。公司采用B2B来料加工模式，按件收费，打样周期4-7天。

总经理李进汉技术出身，对工艺细节和实际回报敏感，决策风格直接。当前公司60人规模，技术团队约8人、品管团队约4人，无专职IT团队，数字化基础为零，决策者对新技术有基本认知。

## 2. 解决方案

### 整体思路

PVD来料加工企业的核心困境：订单靠微信催，批次出问题查不出原因，工艺能力靠嘴说看不见。整个过程缺乏透明度，出了问题只能靠经验猜。

本方案以SaaS为载体，将订单从进到出的全流程串起来，每道工序扫码留记录，质量问题可快速溯源。同时打造PVD工艺能力与案例展示平台，把看不见的工艺优势变成客户能直接看到的图文内容，降低销售沟通成本，帮助获客。展示平台可独立部署，也可与官网结合，作为品牌展示的窗口。

整体分三期推进：一期聚焦订单和追溯，解决最痛的两个问题快速验证价值；二期扩展到生产和设备，打通车间数据；三期加上客户门户和成本分析，同时将展示平台纳入整体获客体系。

### 功能设计

| 功能 | 解决的问题 | 业务价值 |
|------|-----------|---------|
| 订单管理 | 工单从排程、镀膜、质检到出货各环节状态实时可见，订单优先级可设置和调整 | 客户减少催单，跟单员减少重复答复，出货节点可控 |
| 质检记录与批次追溯 | 批次关联工艺参数（真空度、靶材、气体流量、镀膜时间）、设备编号、操作人员，出现质量问题时可快速定位 | 减少批量事故损失，提升客诉处理效率 |
| 扫码追溯 | 每批次/工件绑定二维码，操作人员扫码完成报工和质检记录录入，兼容手机扫码 | 降低一线人员操作门槛，纸质记录习惯可平滑过渡 |
| 生产进度看板 | 日/周视图展示各工单当前工序和预计完成时间，班组长快速识别延误订单 | 主动协调资源，减少被动救火 |
| 设备稼动率监控 | 记录各真空镀膜设备启停时间、运行时长、故障记录，支撑OEE计算 | 识别设备利用率瓶颈，支撑预防性维护 |
| 客户自助查询门户 | 客户通过工单链接查看自己订单当前状态，减少跟单员重复答复 | 提升客户体验，释放跟单工作量 |
| 打样与交期管理 | 打样工单与量产工单分开管理，打样周期4-7天自动预警，量产交期与打样结果挂钩 | 打样进度可控，量产交期承诺有据 |
| 成本核算 | 按订单分工艺类型统计加工费，自动核算毛利 | 快速出具订单成本和利润数据，支撑报价决策 |
| **PVD镀膜能力与案例展示平台** | **无对外展示渠道，客户对工艺能力缺乏直观认知，销售靠嘴说效率低** | **客户点击即看明白工艺效果，减少销售沟通成本，辅助获客转化** |

展示平台功能说明：

- **镀膜工艺展示**：离子镀、磁控溅射、UV电镀等不同工艺的效果对比图文，可按基材类型（不锈钢/铝合金/铜/塑料等）筛选
- **行业应用案例**：渔具、餐具、五金、3C配件等典型行业的合作案例，含产品图、镀膜类型和效果描述
- **工艺能力介绍**：设备类型和数量、可加工尺寸范围、可镀颜色和膜厚范围等参数化展示
- **产品图册**：合作客户产品展示，支持分类检索，可作为对外报价和打样邀约的素材库
- **技术参数页面**：常见工艺问题解答、镀膜原理简介，建立专业认知壁垒

展示平台可独立部署为品牌官网，也可与现有官网整合，作为核心模块嵌入。内容管理后台支持运营人员自主更新案例和图片，无需技术介入。

### 技术方案

| 技术维度 | 方案 |
|----------|------|
| 部署方式 | SaaS多租户架构，按企业订阅，无需本地服务器和维护 |
| 前端交互 | Web端管理后台 + 移动端H5（手机扫码即用，无需安装App） |
| 数据采集 | 条码/二维码标签，兼容PDA手持设备和普通智能手机扫码 |
| 数据存储 | 云端加密存储，批次追溯数据按监管要求保留 |
| 集成能力 | 开放API，支持与财务系统或CRM对接（如有） |
| 展示平台 | 独立部署或与官网集成，响应式设计，支持PC和移动端访问，内容管理后台供运营自主维护 |
| 设备数据采集 | 一期以人工记录为主；如现场确有设备自动化采集需求，可进一步探讨PLC接口对接方案 |

### 差异化优势

| 对比维度 | 现状竞对（Excel+电话+微信） | 通用ERP（用友/金蝶/管家婆） | 本方案 |
|----------|---------------------------|---------------------------|--------|

| 核心能力 | 无 | 大而全 | 订单可视化+批次追溯+工艺展示 |

| 实施周期 | — | 数月 | 一期4-6周起，快速验证价值 |
| IT要求 | 无 | 需专人维护 | 无需专职IT |
| 对外展示 | 无 | 无 | 配套工艺展示平台，辅助获客 |

## 3. 实施路径

### 阶段概览

| 阶段 | 周期（参考） | 核心目标 | 关键交付物 |
|------|------------|----------|-----------|
| 启动 | 约1-2周 | 需求确认，数据准备，环境搭建 | 需求确认备忘，基础数据配置完成 |
| 一期 | 约4-6周 | 订单管理+批次追溯+质检记录+扫码追溯 | 工单全流程在线，批次质量问题可追溯 |
| 二期 | 约3-4周 | 生产看板+设备运行时长+基础OEE | 车间进度可视化，设备利用率数据可见 |
| 三期 | 约4-6周 | 客户门户+成本核算+展示平台+报表分析 | 客户自助查进度，订单成本透明，工艺展示平台上线 |

> 注：各阶段周期为参考值，实际周期需根据现场需求确认情况调整。

展示平台（REQ-012）建议在一期内完成需求调研和原型确认，在三期与客户门户同步上线。展示平台的内容填充（案例拍摄、图文整理）可与系统开发并行推进，建议安排专人负责素材准备。

### 关键里程碑

| 时间节点 | 里程碑 |
|----------|--------|
| 第1-2周 | 需求确认，签订合同，完成基础数据配置（工艺类型、设备列表、人员权限） |
| 第3-6周 | 一期功能开发完成，选取2-3条真实工单试运行 |
| 第7-10周 | 二期功能上线 |
| 第11-16周 | 三期功能上线，展示平台同步发布，完整方案交付 |
